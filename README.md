# 🚀 GSAP과 ScrollTrigger로 만드는 인터랙티브 웹페이지 도전!

프리코스의 자유 주제 미션을 위해 GSAP(GreenSock Animation Platform)을 학습하고, 이를 활용한 도전적인 프로젝트를 2주간 진행합니다.

## 1. 미션 주제

### \*\*GSAP의 핵심기능(Timeline, ScrollTrigger)을 활용한 인터랙티브 가상 브랜트 페이지 제작

단순히 라이브러리를 사용하는 것을 넘어, 사용자의 스크롤과 상호작용에 반응하는 동적인 웹피이지를 구현하여 '도전적인 경험'을 쌓는 것을 목표로 합니다

## 2. 도전 목표

### - 핵심 개념 습득 : GSAP의 Tween, Timeline의 작동 원리를 이해하고 활용한다

### - 플러그인 정복 : GSAP의 가장 강력한 플러그인인 ScrollTrigger를 사용하여 스크롤 기반 애니메이션을 구현한다.

### - 실전 적용 : 학습한 내용을 바탕으로 실제 작동하는 하나의 완성된 웹페이지를 기획하고 구현한다.

### - 문제 해결 : 애니메이션 성능 최적화, 반응형 웹에서의 애니메이션 처리 등 실제 개발에서 마주할 수 있는 문제를 경험하고 해결한다.
